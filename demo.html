Usage

Include the Javascript file.

<script type="text/javascript" src="https://github.com/sunyolis/gbs_cover_art/raw/master/gbs_covers.js"></script>

Next, use the following snippet (with a real ISBN) in the location you want your book's thumbnail to appear.

<div bibkey="ISBN:123456" class="gbookthumbnail"></div>

OR, for a thumbnail that links to additional information on Google Books

<div bibkey="ISBN:123456" class="gbookpreview"></div>
